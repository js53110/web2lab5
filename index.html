Pritisni plavi gumb za pretplatu na obavijesti

Demonstracija PWA

Implementirano:

  • Subscribe - pritiskom na tipku korisnik se pretplacuje na obavijesti
  • Caching - app shell se cache-ira (index.html, app.js, fallback stranica)
  • Dynamic caching - sve stranice koje se posjete se cache-iraju
  • Camera api
  • Fallback metoda - ako kamera nije dostupna ili je blokirana - omogucen upload slike
  • Background sync - ukoliko se uslika ili uploada slika dok internet nije dostupan, sprema se u IndexedDB i sinkronizira se nakon ponovnog spajanja na internet
  • Slikanjem/uploadanjem slike okida se background sync te se salje push obavijest korisnicima koji su pretplaceni
